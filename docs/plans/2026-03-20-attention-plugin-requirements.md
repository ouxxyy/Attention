## 1. 项目概述

### 1.1 产品定位

**产品名称**: 注意力助手 (Attention)
**产品类型**: 浏览器插件 (Chrome Extension) + 移动端配套

**核心价值主张**: 帮助用户量化和管理注意力资源，通过浏览器使用行为分析提供实时注意力状态反馈和AI驱动的优化建议。

### 1.2 核心问题

- 人类注意力是有限资源
- 任务切换会导致注意力快速衰减
- 持续工作也会逐渐消耗注意力
- 用户缺乏量化工具了解自身注意力状态

### 1.3 目标用户

| 用户群体   | 核心需求             | 付费意愿 |
| ---------- | -------------------- | -------- |
| 知识工作者 | 提升专注力、工作效率 | 中等     |
| 学生       | 备考学习、减少分心   | 较高     |
| 远程办公者 | 工作生活平衡、自律   | 中等     |
| 创意人士   | 保护深度工作时间     | 高       |

## 4. 用户体验

### 4.1 核心用户流程

#### 首次使用流程

1. 用户安装插件 → 引导页介绍功能
2. 创建账户 (可选) → 选择订阅方案
3. 设置工作时间段 → 授权必要权限
4. 开始使用 → 显示实时注意力分数

#### 日常使用流程

1. 打开浏览器 → 插件自动开始追踪
2. 切换标签页/应用 → 系统记录行为
3. 注意力分数变化 → 实时显示在插件图标
4. 分数过低 → 发送温和提醒
5. 查看报告 → 访问仪表盘

### 4.2 界面设计原则

- **简洁**: 保持界面干净，避免信息过载
- **温和**: 提醒措辞友好，不制造焦虑
- **渐进**: 功能逐层展示，不强迫用户
- **实用**: 突出核心指标，减少装饰

### 4.3 关键界面

| 界面     | 核心元素                         |
| -------- | -------------------------------- |
| 插件弹出 | 注意力分数圆环、趋势图、快捷操作 |
| 仪表盘   | 分数趋势、热点分析、建议卡片     |
| 设置页   | 账户信息、通知设置、订阅管理     |
| AI分析页 | 深度报告、个性化建议、训练计划   |
